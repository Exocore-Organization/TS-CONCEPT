# Skills Index

Use `/ts-<name>` to load a skill:

| Command | Description |
|---------|-------------|
| `/ts-philosophy` | Core philosophy: no `any`, no `var`, no legacy. Banned vs replacement table. |
| `/ts-patterns` | Code patterns: branded types, discriminated unions, `satisfies`, `as const`, `using`. |
| `/ts-jsdoc` | JSDoc standards: `@param`, `@returns`, `@throws`, `@example` on every export. |
| `/ts-config-tooling` | `tsconfig.json`, `tsx`, `tsc --noEmit`, project structure. |
| `/ts-ai-behavior` | AI agent rules: reject/require/enforce/prefer/verify. |
| `/ts-mcp-config` | MCP server setup — tools for `ts_typecheck`, `ts_read_skill`, `ts_list_skills`. Works with OpenCode, Gemini, Codex, Exocode, Hermes. |
